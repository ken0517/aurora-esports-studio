# Aurora Placeholder Assets

These files are generated by `npm run assets` and are safe to replace before launch.

- `brand/aurora-dashboard.png` - hero dashboard visual placeholder
- `results/result-01.png` to `results/result-04.png` - rank result / record screenshot placeholders
- `reviews/review-01.png` to `reviews/review-03.png` - testimonial screenshot wall placeholders

Keep the same filenames to update the live site without changing code, or update `src/data/content.js` if you prefer different paths.
