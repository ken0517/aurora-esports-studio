# KLG Studio 免费 AEO 与港澳台服务范围设计

## 一、目标

本次工作同时处理两件彼此相关、但目的不同的事项：

1. 将 KLG Studio／Aurora Esports Studio 的真实服务范围由香港及台湾扩展为香港、台湾及澳门。香港仍是主要市场，澳门是实际新增可接待的市场，不是为了增加 AEO 关键词。
2. 建立一套不依赖付费 HubSpot 的人工 AEO 追踪方法，用 25 个真实顾客问题观察 KLG Studio 在 ChatGPT、Gemini 及 Perplexity 的品牌出现、引用和描述变化。

本次不制作 YouTube 内容或教学文章；这些内容以后如有需要再独立规划。

## 二、公开品牌与市场表达

所有顾客可见内容使用一致事实：

> KLG Studio 以香港为主要市场，同时为台湾及澳门玩家提供线上游戏服务，支持繁体中文、简体中文及英文查询。

首页短句采用「香港为主・台湾及澳门均可查询」的含义；英文市场标示由 `HK · TW` 扩展为 `HK · TW · MO`。

地区名称必须按语言正确显示：

- 繁体中文：香港、台湾、澳门
- 简体中文：香港、台湾、澳门
- 英文：Hong Kong, Taiwan, Macau

现有 WhatsApp、LINE、Instagram、Discord 及 Carousell 联系渠道继续保留。网站不假设澳门顾客只使用某一个平台；香港仍建议 WhatsApp，台湾仍可使用 LINE，三个地区都可以使用其他公开渠道查询。

## 三、中央资料与搜索资料

地区和语言不再散落为互相矛盾的固定文字。公开品牌资料集中保存：

- 主要市场：香港
- 可服务市场：香港、台湾、澳门
- 支持语言：繁体中文、简体中文、英文
- 主要服务品牌：KLG Studio
- 官方网站身份：Aurora Esports Studio

网站公开页面、首页、游戏服务页、FAQ、页尾、SEO 标题和描述、AI 可读文字以及产生公开页面的程序，都读取或遵循同一组资料。

结构化资料会把服务地区列为 Hong Kong、Taiwan、Macau，并在联络资料中说明可使用 Traditional Chinese、Simplified Chinese 及 English。网站没有可供顾客到访的实体地址，因此不会伪装成有门市的本地商户，也不会虚构 Google 商家地址。

## 四、真实 Carousell 评价

用户提供的截图可确认：

- 公开账号：`@klg_studio`
- 评分：5.0／5
- 公开评价数：30 条
- 截图中可核对的评价包括：
  - 「专业、快手快脚。」
  - 「good 快手」
  - 「回复快、有效率。」
  - 「下次要再找你帮忙，回复快、效率高，Nice。」

公开网站采用以下原则：

- 在 KLG Studio 品牌页加入简洁的「真实评价」区段。
- 显示 `5.0／5 · 30 条 Carousell 公开评价`，并注明资料来自 KLG Studio 的公开 Carousell 页面。
- 评价文字使用忠于原意的繁体、简体及英文显示版本；不会改变评价的正负含义。
- 不公开买家账号、头像或其他可识别资料。
- 原始截图只作为核对证据，不直接放进公开网站，避免暴露顾客资料，并让 AI 能读取文字而不是只看到图片。
- 截图显示的是账号在 Carousell 上的年期，不会擅自写成 KLG Studio 已经营相同年数。
- 不使用「香港第一」「绝对安全」「保证胜率」等无法由评价证明的宣传。

评价数字属于时间点资料。中央资料会保留最后核对日期，日后数量变化时可以集中更新。

## 五、25 个免费 AEO 追踪问题

建立一份内部追踪表，而不是要求网站顾客回答问题。问题分配如下：

- 香港市场：12 个
- 台湾市场：5 个
- 澳门市场：3 个
- KLG 品牌、评价与竞争比较：5 个

问题覆盖以下顾客意图：

- 推荐香港、台湾或澳门的传说对决／王者荣耀／HOK 游戏服务
- 排位、陪玩、巅峰赛、英雄战力标及教学需求
- 如何判断工作室是否可靠
- KLG Studio 与 Aurora Esports Studio 的关系
- KLG Studio 的公开评价及联络方式
- KLG Studio 与斗士工作室等公开竞争对象在 AI 答案中的出现情况

斗士工作室只作为内部追踪竞争对象。网站不会发布未经核实的攻击、排名或直接优劣结论。

每次检查记录：

- 检查日期和 AI 平台
- 完整问题
- 是否提及 KLG Studio
- 是否附上 `https://auroraesportstudio.com/`
- 品牌出现位置
- 描述为正面、中立、负面或错误
- 同时出现的竞争品牌
- AI 引用的网页和平台
- 需要采取的下一步

第一版建立空白基准表和明确测试方法，不伪造 ChatGPT、Gemini 或 Perplexity 的答案。实际答案必须在对应平台测试后逐次记录。

## 六、三十日执行方式

1. 发布地区、语言与真实评价更新。
2. 建立 25 个问题的初始基准。
3. 每星期使用相同问题检查 ChatGPT、Gemini 及 Perplexity。
4. 比较 KLG Studio、Aurora Esports Studio 和斗士工作室的出现与引用来源。
5. 只根据真实数据决定是否新增 FAQ、服务说明或以后制作教学文章。

HubSpot AEO 暂不购买。若未来人工记录量过大，再独立评估自动化工具。

## 七、不在本次范围

- YouTube 频道或影片
- 教学文章与博客系统
- 付费 HubSpot AEO
- Google Ads
- 伪造评价、外部文章或第三方推荐
- 公开攻击或贬低竞争对手
- 修改价格、报价规则、AI 客服逻辑或后台订单系统
- 设置实体门市或虚构地址

## 八、验收条件

- 所有主要公开页面不再把服务范围限制为香港及台湾。
- 首页清楚说明香港为主要市场，并真实包含台湾和澳门。
- 三种语言都正确包含澳门市场。
- 结构化资料的服务地区和语言与页面可见内容一致。
- KLG Studio 页面显示匿名、可核对的真实 Carousell 评价。
- 公开页面不包含买家账号、头像或私人资料。
- 25 个问题及追踪字段完整建立，市场分配合计为 25。
- 斗士工作室只出现在内部追踪资料，不作为公开攻击内容。
- 原有价格、报价、客服、后台和手机布局没有被破坏。
- Tests、Lint 及 Production Build 全部通过。
