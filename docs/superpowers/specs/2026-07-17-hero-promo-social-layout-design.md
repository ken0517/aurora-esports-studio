# Aurora 首頁優惠與聯絡按鈕調整設計

## 目標

改善手機首頁首屏的資訊層級，讓新人優惠與品牌直接關聯，並確保五個社交平台按鈕不被底部報價入口遮擋。

## 已批准設計

- 將「新人優惠・全單自動享 85 折」放在頂部中央 Aurora Esports Studio 品牌名稱正下方。
- 優惠文字使用現有金色系統及細小字級，維持精品感，不使用醒目的電商膠囊樣式。
- 完全移除首頁首屏的支付寶、微信支付、轉數快圖案及付款說明。
- 手機版五個社交平台按鈕整組向上移，與底部兩個報價入口保持清楚間距。
- 電腦版只保留必要的位置調整，不改動其他首頁內容、品牌、報價功能或社交連結。

## 響應式規則

- 優惠文字成為品牌標誌的一部分，在桌面與手機均置中顯示。
- 手機版社交按鈕位於報價入口上方，保持可點擊、可讀及不重疊。
- 小屏幕寬度不足時仍允許現有換行機制，不造成水平捲動。

## 驗收

- 原有付款圖案及付款說明不再出現在首頁 DOM。
- 優惠文字出現在品牌名稱內部結構下方。
- 手機 CSS 將社交按鈕提升至報價入口上方。
- 現有社交連結、手動報價及 Aurora 客服入口保持可用。
- Tests、Lint、正式版 Build 通過，並以手機及電腦畫面覆核。
