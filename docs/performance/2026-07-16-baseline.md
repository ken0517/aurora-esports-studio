# Aurora 首頁速度優化紀錄

## 優化前（2026-07-16）

- 首頁單一 JavaScript：509.03 kB（gzip 159.81 kB）
- 全站樣式：62.78 kB（gzip 13.16 kB）
- 電腦版 Hero 圖片：2,572,688 bytes
- 手機版 Hero 圖片：2,607,836 bytes
- 兩張 Hero 圖片合計：5,180,524 bytes
- 三張遊戲服務圖片合計：5,172,629 bytes

## 優化後（2026-07-16）

- 首頁入口 JavaScript：193.43 kB（gzip 61.35 kB）
- 首頁程式、翻譯及資料等首次載入合計：約 421.55 kB（gzip 約 138.49 kB）
- 報價助手：71.28 kB（gzip 20.17 kB），只在客人點擊後載入
- 後台管理：與首頁分開載入，不再增加一般客人的首頁下載量
- 首頁樣式：35.33 kB（gzip 7.82 kB）
- 報價助手樣式：14.70 kB（gzip 3.68 kB），只在使用報價功能時載入
- 電腦版 Hero WebP：365,656 bytes
- 手機版 Hero WebP：361,070 bytes
- 兩張 Hero 圖片合計：726,726 bytes，減少 85.97%
- 三張遊戲服務 WebP 合計：566,742 bytes，減少 89.04%
- 首頁入口 JavaScript 原始大小減少約 62.00%，gzip 大小減少約 61.61%

以首次開啟首頁的主要資源估算，傳輸量由約 5.35 MB 降至約 0.87 MB，減少約 83.7%。實際速度仍會受手機、網絡、瀏覽器快取及伺服器距離影響。

## 已採用方式

- 電腦版與手機版 Hero 分別使用對應比例的 WebP 圖片。
- 遊戲服務圖片轉為高品質 WebP，並保留延遲載入。
- 報價助手、營運後台及非首頁功能分開載入。
- 首頁先顯示輕量的「手動報價／Aurora 客服」入口，客人點擊後才下載完整報價功能。
- 保留原有視覺、資料及報價規則，沒有改變業務內容。
